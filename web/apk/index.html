Cara install:
  1. Tap tombol Download APK di atas
  2. Kalau Chrome bilang "file ini mungkin berbahaya" — tap "Tetap Simpan" / "Keep"
  3. Buka file APK yang baru di-download
  4. Izinin "Install dari sumber tidak dikenal" kalau diminta
  5. Tap Install → Buka
  6. Pertama kali buka, izinin akses Kamera + Mikrofon (buat voice/video chat)
